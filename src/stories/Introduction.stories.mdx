# Tint themes lib

#### To be able to use the components you have to create the Tint and install: <a href="https://www.npmjs.com/package/@tintup/tint-sdk" target="_blank">tint-sdk</a> After that you can install the <a href="https://www.npmjs.com/package/@tintup/tint-themes" target="_blank">themes lib</a> and build your own theme from our components.
